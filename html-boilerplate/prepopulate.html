Ordered List of Procedures for Making Webapp to Calculate Space Groups

  1. Type in a number, checks against a list of numbers to see if it's in the list.
  2. Input 2 numbers, the app divides one number with the other, checks if the result is on a list with ratios and corresponding names, returns the corresponding name.
  3. Input a txt file with wavenumbers and intensity values, returns a logarithmic plot.
  4. Input a txt file, returns the peak maxima.
  5. Input a txt file, with wavenumbers and intensity values. Returns a plot of wavenumbers against intensity values, a list of peak maxima, and the spacegroup, which is obtained through calculating the ratios between the maxima.
